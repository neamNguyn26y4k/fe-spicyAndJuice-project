# fe-spicyAndJuice-project

## Hướng dẫn mở website

1. **Clone code về máy:**

   ```bash
   git clone https://github.com/neamNguyn26y4k/fe-spicyAndJuice-project.git
   ```

2. **Mở thư mục dự án trong VS Code hoặc trình soạn thảo bạn muốn.**

3. **Mở file:**

   - Vào thư mục `public/`
   - Mở file `home.html` bằng trình duyệt (có thể kéo thả file vào trình duyệt hoặc click chuột phải chọn "Open with" > trình duyệt bạn muốn).

4. **Trải nghiệm website!**
